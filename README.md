# Titanic Dataset Analysis

This project performs a **data analysis of the Titanic dataset** using Python to identify if there is any **probability of survival** based on different variables.

---

## Overview

The analysis aims to answer the following question:

- Is there any **relationship** between specific variables (e.g., age, gender, ticket class) and the likelihood of **surviving the Titanic accident**?

---

## Tools & Methods

- **Python** for data analysis and visualization.
- Statistical methods and machine learning for probability estimation.
